# 🏥 CathLab — سیستم مدیریت نسخه اتاق عمل کت‌لب

سیستم جامع مدیریت نسخه و صف انتظار بیماران برای اتاق عمل کتتریزاسیون قلبی (کت‌لب) بیمارستان گلستان اهواز.

---

## ✨ امکانات

### بیمار
- ارسال آنلاین نسخه با آپلود تصویر یا PDF
- پیگیری نوبت با **کد ملی** یا **نام و نام خانوادگی**
- نمایش موقعیت در صف، وضعیت بستری، و نتیجه عمل
- امکان ثبت درخواست اورژانسی

### پنل مدیریت
- **داشبورد** آماری به تفکیک هر پزشک
- **تب لیست بیماران** — تأیید، تغییر وضعیت، بستری کردن دسته‌جمعی
- **تب اورژانسی** — تأیید یا رد درخواست‌های اورژانسی
- **تب بستری** — مدیریت بیماران بستری امروز و تاریخچه
- **تب انجام شده‌ها** — ثبت نتیجه عمل (نرمال / PCI / درمان دارویی / مشاوره جراحی قلب باز)
- **تب کاربران** — مدیریت کامل کاربران با نقش‌های مختلف

### مدیریت کاربران
- احراز هویت با یوزرنیم و پسورد
- نقش‌های: مدیر کل / پزشک / بیننده بستری
- هر پزشک فقط بیماران خود را می‌بیند
- افزودن، ویرایش، فعال/غیرفعال و حذف کاربر

### صف انتظار عمومی
- صفحات نمایش صف هر پزشک به‌صورت مجزا
- صفحه مجموع کل صف با نمودار درصد هر پزشک
- بروزرسانی خودکار هر ۳۰ ثانیه
- **PWA** — قابل نصب روی iOS و Android

### ثبت نسخه دستی
- ورود با رمز ادمین
- انتخاب تاریخ ثبت از **تقویم شمسی** داخلی

## 📁 ساختار فایل‌ها

```
cathlab/
├── index.php           # فرم ارسال نسخه + پیگیری
├── submit.php          # پردازش فرم ارسال
├── submit-manual.php   # ثبت نسخه با تاریخ دستی
├── admin.php           # پنل مدیریت کامل
├── config.php          # اتصال دیتابیس + ساخت جداول
├── queue.php           # صف انتظار هر پزشک
├── queue-all.php       # مجموع صف همه پزشکان
├── queue-manifest.json # PWA manifest
└── uploads/            # فایل‌های آپلودشده
```

## ⚙️ نصب

**۱. دیتابیس**

در cPanel یک دیتابیس MySQL بساز و اطلاعات را در `config.php` وارد کن:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'your_db_name');
define('DB_USER', 'your_db_user');
define('DB_PASS', 'your_password');
```

جداول به‌صورت خودکار در اولین بارگذاری ساخته می‌شوند.

**۲. آپلود**

فایل‌ها را در `public_html` یا یک زیرشاخه آپلود کن.

**۳. دسترسی‌ها**

```bash
chmod 755 uploads/
```

**۴. ورود اولیه**

|---|---|

> ⚠️ پس از اولین ورود حتماً رمز را از تب کاربران تغییر دهید.
